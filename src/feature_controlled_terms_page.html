sCubed uses controlled vocabularies to help promote consistency and clarity.

Attention! You should not add terms to drop-down lists by clicking the pencil (Edit button) that Google displays in the drop-down lists. Drop-down lists should only be updated by entering information into the sheets and then selecting Refresh data validation from the sCubed menu.

Below is a list of the columns with drop-down lists that sCubed does not intend for you to edit, meaning these lists cannot be updated with the item Refresh data validation. These lists include keywords that support the functionality of sCubed.
