About

Share Science with Sheets (sCubed) is a spreadsheet applicationt that enables researchers to capture information about how scientific experiments were conducted.

Start by configuring the sCubed forms. Navigate to the sCubed menu and then navigate to the item Configure forms.
  1. Create the core sCubed Sheets. Select 1. Create core forms
  2. Choose whether you want to summarize the provenance of physical material or describe individual activities. Then, select either 2A. Enable summaries or 2B. Enable activities.
  3. Add columns that are relevant to your context. Select 3. Add context packages and then complete the form.
  4. Add data validation and formatting to core sheets. Select 4. Apply core validation.
  5. Add data validation and formatting to activity sheet. Select 5. Apply activity validation.
  6. Install the onEdit trigger to enable the dynamic responses to your edits. Select Install onEdit trigger.
